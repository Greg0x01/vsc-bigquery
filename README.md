# BigQuery for Visual Studio Code

## Shortcuts

| Shortcut | Effect |
|----------|--------|
| `CTRL-Enter` | Submit query |
| `CTRL-Shift-Enter` | Submit region | 
| `CTRL-Alt-Enter` | Submit query for dry run |
